# earth-leyu-resource

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面资源。这些页面可能来自不同的来源或用途，它们被集中收集于此，以便于长期保存和访问。仓库不针对任何特定域名、网站或服务，仅作为静态页面文件的存储与分发点。

## 目录说明

仓库根目录下可能包含以下类型的内容：

- **`/pages`**：存放主要的 HTML 页面文件及其相关资源（如 CSS、JavaScript、图片等）。
- **`/archives`**：历史版本或不再更新的页面归档。
- **`/assets`**：公共资源文件，例如样式表、脚本库或字体文件。
- **`/docs`**：额外的说明文档或使用指南（可选）。

每个子目录内可能附带独立的 `README` 文件，用于描述该目录下页面的具体用途或来源。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务或数据库。
- 页面可能包含外部资源（如 CDN 链接），但尽量保持自包含性。
- 归档的页面可能来自过往项目、临时页面、测试用例或其他公开来源。
- 部分页面可能包含过时信息或不再维护，仅作历史记录存档。

## 维护说明

- 本仓库由维护者不定期更新，不保证实时同步或持续发布。
- 欢迎通过 Issue 提交问题或建议，但不保证及时响应。
- 不接受直接修改仓库内容的 Pull Request，除非用于修复明显错误或安全隐患。
- 所有内容仅供存档参考，请自行判断适用性。

## 许可

除非特定目录或文件另有说明，本仓库内容按原样提供，不附带任何明示或暗示的保证。使用者应自行承担使用风险。
